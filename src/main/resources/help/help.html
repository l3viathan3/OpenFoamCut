OpenFoamCut User Guide

Version: 1.1

Welcome to OpenFoamCut

OpenFoamCut is a desktop application for preparing cutting jobs on 4-axis CNC hot-wire foam cutters. This guide covers the complete workflow from loading profiles to generating G-code for your machine.

Basic Workflow

The application is organized into five main tabs that guide you through the cutting job preparation:

  1. 1. File Select & Arrange — Load outline files and position them on the foam block
  2. 2. Additional Geometry & Cut Order — Define cut sequences and synchronization points
  3. 3. Machine Setup — Configure G-code templates and machine profiles
  4. 4. Generate G-Code — Create and export the final cutting code
  5. 5. 3D Preview — Inspect the generated shape as a rotatable 3D model

The Help tab provides this reference guide.

Supported File Formats

Input Files

  • .dxf — DXF outline files containing cutting profiles
  • .dat — Airfoil/profile files in Selig or Lednicer format

Important: Profile files must contain exactly one closed outline (one "island") of connected lines. Files with multiple separate outlines or isolated lines will be rejected.

Output Files

  • .ofc — OpenFoamCut project files that save your work session for later editing
  • .gc — G-code files ready for CNC machine execution

Menu Options

File Menu

  • Open (Ctrl+O) — Load a previously saved .ofc project file
  • Save (Ctrl+S) — Save the current project as an .ofc file
  • Exit (Ctrl+Q) — Close the application and save the current state

View Menu

  • Light Theme — Switch to a light color scheme
  • Dark Theme — Switch to a dark color scheme

Tab 1: File Select & Arrange

Use this tab to load outline files, define your foam block dimensions, and position the outlines for cutting. The working area displays live previews of the left and right outlines positioned on the foam block.

A. Outline Files

  • Select Left File — Click to browse and choose a .dxf or .dat file for the left outline
  • Select Right File — Click to browse and choose a .dxf or .dat file for the right outline
  • Swap Left and Right — Quickly exchange the left and right file selections

The width and height dimensions of each loaded outline are displayed automatically.

B. Foam Material

I. Foam Block Dimensions

  • Height — The vertical dimension of your foam block
  • Width — The horizontal dimension of your foam block
  • Depth — The thickness of your foam block (front to back)

As you enter dimensions, the application automatically checks whether your outlines fit within the foam block and displays a message indicating fit status.

II. Foam Position

  • X offset from origin — Shifts the entire foam block along the X axis relative to the machine origin
  • Offset from left wire anchor — Sets the spacing between the foam block and the left-side wire anchor point
  • Offset from right wire anchor — Sets the spacing between the foam block and the right-side wire anchor point

C. Outline Placement

Left Outline & Right Outline Positioning

Each outline can be positioned independently using two methods:

  • Arrow Buttons — Fine-tune position with directional controls. Hover over buttons to see which direction and magnitude they adjust.
  • Direct Entry — Set precise X and Y offset values numerically in the text fields labeled "Offset X" and "Offset Y"

The preview on the left side of the screen shows both outlines as they will be positioned on the foam block.

Outline Geometry Adjustments

  • Rotation — Rotate each outline independently around its center point (in degrees)
  • Scale — Scale each outline larger or smaller while maintaining proportions

Outline Angles (Foam Orientation)

  • Front Edge (deg) — The angle of the wire at the front of the foam block relative to the machine origin
  • Back Edge (deg) — The angle of the wire at the back of the foam block relative to the machine origin

These angles control the taper or twist of the cut. If both angles are equal, the cut will be parallel (untapered). Different angles create a wedge or twisted shape.

Preview Area

The preview displays a top-down view of your foam block with the left and right outlines positioned on it. The red circle marks the machine origin point. Use this to verify your dimensions, positions, and angles are correct before proceeding to cut order configuration.

Tab 2: Additional Geometry & Cut Order

After arranging your outlines, use this tab to define how the cutting will proceed. You can add additional geometry, define synchronization points between left and right sides, and establish the sequence of the cut.

Arrange Lock

The Lock Arrange checkbox controls access to Tab 1 (File Select & Arrange) while you work on Tab 2. Locking is required before you can generate G-code.

  • Lock Arrange — After finalizing your outline positions in Tab 1, enable this checkbox. When locked, you cannot change outline files, rotation, or scale. However, you can still move outlines, configure foam settings, and swap outlines. All additional geometry you add (lines, sync points, cut order) will move together with outline adjustments.

Unlocking: If you unlock later, all additional geometry (lines, sync points, and cut order) will be deleted, but you regain the ability to edit outline files, rotation, and scale.

A. Additional Geometry

Add extra lines to your cutting pattern independent of the main outline. This is useful for internal cut paths, reinforcement lines, or guide marks.

Adding Additional Lines

  1. Click the Add Line button for the side (left or right) where you want to add a line
  2. Click your desired start point on the outline display — a message will prompt you to select the start point
  3. Click your desired end point on the outline display — the line is automatically added when you select the second point

Managing Additional Lines

  • List of Additional Lines — Shows all lines you have added for each side
  • Remove Selected — Delete a highlighted additional line from the list

You can add multiple lines on both the left and right sides. This is useful for creating lead-in paths from the origin point to your main outlines, or for adding internal cut patterns.

B. Synchronized Control Points

Synchronization points define where the left and right wire anchors should meet at the same moment. This ensures proper alignment during the cut.

Adding Synchronization Points

  1. Click the Add Sync Point button to start the point selection process
  2. Click your desired point on the left outline display — a message will prompt you to select the left point
  3. Click the corresponding point on the right outline display — the synchronized pair is automatically created when you select the second point

Managing Synchronization Points

  • List of Synchronization Points — Displays all synchronized point pairs you have created, with coordinates for both left and right sides
  • Remove Sync Point — Select a point pair in the list and click this button to delete it
  • Highlighting — When you select a sync point in the list, the corresponding points are highlighted with markers on both the left and right outline displays to help verify your setup

C. Cut Order Sequence

Define the order in which synchronized points are reached during the cut. The G-code generator will travel along outlines and additional lines, stopping at each sync point in your cut order sequence.

Essential Rules for Cut Order

  • Always start and end at the origin: Your first and last sync point must be at the machine origin (0, 0). This ensures the wire anchors return to their home position.
  • Untraveled paths are preferred: When moving between two sync points, the machine will prefer to travel along paths that have not yet been traversed.
  • Path selection: If multiple untraveled segments are available, the upper path is preferred.

Cut Order Controls

  • Add to Cut Order — Add a highlighted synchronization point to the sequence
  • Move Up / Move Down — Adjust the order of points in the cut sequence
  • Remove from Cut Order — Delete a point from the sequence

Example: Aircraft Wing Cutting

A typical wing cutting setup includes:

  • Outlines: Left and right airfoil profiles representing the wing shape
  • Additional Lines: On both left and right sides, click Add Line and select two points from origin (0, 0) to the tip of each airfoil outline. This ensures the wire can travel from the origin point to start cutting the airfoil.
  • Sync Points: Define at least three pairs by clicking Add Sync Point and selecting the corresponding points on each side:
    • Origin point (0, 0) on both sides
    • Tip of airfoil (where the additional line meets the outline) on both sides
    • End of airfoil on both sides
  • Cut Order Sequence: Then in section C, add these sync points to the cut order in this sequence: origin → tip → end → tip → origin

    This sequence makes the wire travel: origin to tip (along the additional line), tip to end (along the outline), end back to tip (along the outline via untraveled path), and tip back to origin (along the additional line).

Preview Area

The preview displays the left and right outlines with any additional geometry and synchronization points visible. Points are shown as markers; verify your cut path routing is correct, especially that untraveled segments exist between your sync points and that the sequence creates a valid cutting path before generating G-code.

Tab 3: Machine Setup

Configure your CNC machine parameters and create reusable machine profiles for different cutting jobs.

A. Profiles

Machine profiles store your G-code templates and settings so you can quickly switch between different machine configurations.

Profile Management

  • Profile Dropdown — Select a saved machine profile
  • Load — Load the selected profile into the editor below
  • Profile Name — Enter the name for a new profile
  • Save As — Save the current G-code templates and settings under a new profile name
  • Delete — Remove a selected profile

B. G-Code Templates

Define the code that runs at the start, during each line move, and at the end of the cutting sequence. The template system allows you to customize output for your specific machine.

Start Code

G-code commands executed once at the beginning of the job. Typically includes machine initialization, spindle speed setup, or positioning commands.

Line Code

A template for each cutting line output during the job. Use placeholders to reference synchronized position data (coordinate details depend on your machine type).

End Code

G-code commands executed once at the end of the job. Typically includes spindle stop, machine park position, or shutdown commands.

Information Section

The hints area displays guidance on using the template system and available placeholders for your machine type.

Tab 4: Generate G-Code

Generate the final G-code and review it before saving to a file for machine execution.

A. Generation and Preview

  • Generate — Create G-code based on your current outline, cut order, and machine setup
  • Generate and Save — Create G-code and immediately save it to a .gc file
  • Play/Pause Simulation — Preview the cut path with an animated visualization

B. G-Code Output

The text area displays the generated G-code. You can review the commands before saving. Copy text directly from this area if you need it elsewhere.

C. Left and Right Previews

The left and right preview panels show the actual cut path that will be followed by each wire. The red circle marks the starting anchor point for each side. Use these previews to verify the cut geometry is correct before running the job on your machine.

D. Sanity Checks

When you generate G-code, OpenFoamCut automatically performs validation checks on your job configuration. These checks warn you of potential problems before you save the output for machine execution. The check results appear in the panel to the left of the G-Code generation buttons:

  • Outlines fit within foam block — Verifies that both the left and right cutting outlines fit within the foam block dimensions you specified. If an outline extends beyond the foam, you'll need to adjust the outline position, the foam dimensions, or both.
  • Wire anchors stay within foam block dimensions — Checks that the wire anchor motion points (entry and exit positions) remain within the foam block. This ensures the wires can physically reach all positions during the cut.
  • All anchor points have non-negative coordinates — Validates that anchor points never move into negative X or Y values. Negative coordinates would place the anchors before the machine origin or below the table, which is not physically possible.
  • All main and additional lines are properly connected — Ensures there are no isolated "islands" of disconnected geometry in your outline or additional lines. All cut segments must form a single connected path or set of connected components.
  • G-Code will cut through all main and additional lines — Verifies that your cut sequence (defined in Tab 2) actually visits and cuts all the line segments you've specified. If additional lines are defined but not reachable from your sync points, this check will alert you.
  • Foam dimensions are valid — Confirms that your foam block dimensions (width, depth, height) are all positive numbers. Zero or negative dimensions indicate incomplete setup.
  • Outlines are not empty — Checks that both left and right outlines contain actual line segments. Empty outlines suggest a file loading or geometry problem.

All checks must pass before saving G-code. If any check shows a warning or failure, review the message and correct the underlying issue in the appropriate tab (file arrangement, cut order, or machine setup) before generating again.

Tab 5: 3D Preview

Use this tab to inspect the currently configured cut as a 3D model. When you open the tab, OpenFoamCut regenerates the preview from the current outlines, cut order, and foam dimensions. If the current setup does not produce a valid closed geometry, the tab shows an error message instead of a model.

A. What the Preview Shows

  • Foam block — The foam dimensions and offsets from Tab 1 are shown as a 3D box
  • Cut shape — The left and right cut outlines are connected across the foam width to show the resulting part shape
  • Machine table — A simple table outline is included to give spatial context for the foam placement

Use this view to confirm the part thickness, taper, and overall orientation before exporting or running the job.

B. Geometry Modes

  • Approximate Mesh — Displays the result as a wireframe mesh between the left and right cut profiles
  • Closed Solid — Displays the result as a closed shaded solid so you can inspect the final volume more easily

C. Mesh Step

The Mesh step control adjusts how densely the preview samples the cut geometry.

  • Smaller values — Create a denser mesh with more cross-sections and a more detailed wireframe preview
  • Larger values — Reduce the number of sampled sections for a simpler, lighter preview

Changing the mesh step immediately rebuilds the 3D preview using the new sampling distance.

D. Mouse Controls

  • Left mouse drag — Rotate the view
  • Mouse wheel — Zoom in and out
  • Double-click — Reset the view to the default angle

If the tab reports that no valid geometry is available, return to Tabs 1 through 4 and fix the outlines, sync points, cut order, or other validation issues before reopening the preview.

Project Files (.ofc)

OpenFoamCut projects are saved in .ofc format. A project file stores:

  • Loaded outline file paths and positions
  • Foam block dimensions and positioning
  • Additional geometry and cut order sequence
  • The current application state for quick recovery on restart

Note: Machine profiles are saved separately and not included in project files. This allows you to share projects while keeping machine settings specific to each user.

Tips and Best Practices

Profile Validation

When loading a profile file, OpenFoamCut checks that it contains exactly one island (one continuous closed outline). If your file has multiple separate shapes or open paths, the application will reject it with an error message. Ensure your outline files are properly closed and connected.

Positioning Workflow

  1. Load both left and right outlines in Tab 1
  2. Set foam block dimensions
  3. Adjust outline positions and rotations until the preview shows the correct geometry
  4. Set foam positioning offsets for your machine
  5. Configure edge angles for any taper or twist
  6. Move to Tab 2 to define additional lines, sync points, and the cut order after locking down Tab 1

Cut Order Configuration

For synchronized cutting, always define sync points on matching features of both outlines. The cut order sequence must start and end at the origin, and you should strategically place sync points so that the machine can follow untraveled paths between them.

When designing cut order sequences, remember:

  • Additional lines from the origin to key outline features (like wing tips) enable efficient routing between distant sync points
  • A well-designed sequence minimizes backtracking by ensuring untraveled paths exist between consecutive sync points
  • If the left and right geometries are very different, you may need multiple sync points to maintain proper wire tension throughout the cut
  • Always verify in the preview that your sync point sequence creates a continuous cutting path

Machine Setup

Create and save profile templates for each machine or cutting style you use. This greatly speeds up project setup for repeated work.

3D Verification

Open Tab 5 after your cut setup is complete to verify that the left and right profiles form the expected 3D part. This is especially useful for tapered or twisted wings, where a 2D preview is not enough to catch orientation mistakes.

Saving Your Work

Use File → Save regularly to preserve your project configuration. The application also automatically saves your session state when you exit, so your work is not lost if the application closes unexpectedly.

Additional Information

For more details on DXF and DAT file formats, consult the format documentation for your CAD or profile design software. For machine-specific G-code template help, refer to your CNC machine's programming manual.
